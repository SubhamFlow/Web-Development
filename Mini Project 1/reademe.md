# Mini Project - Node.js Backend

A simple full-stack app built with Node.js, Express, MongoDB and EJS while learning backend development.

## Features
- User registration and login
- JWT authentication with cookies
- Password hashing with bcrypt
- Create and view posts
- Protected routes with middleware

## Tech Stack
- Node.js / Express
- MongoDB / Mongoose
- EJS (templating)
- JWT
- Bcrypt
- Cookie Parser
- Tailwind CSS
