
# SLIDE.AI 🚀

This project is a high-speed, AI-powered engine designed to transform a single text prompt into a professional, multi-slide PowerPoint presentation. It integrates real-time LLM processing with dynamic media fetching and client-side document generation.

🛠️ Technologies & Tools

Component - Technology
LLM Engine - Groq API (Llama 3.3 70B Versatile)
Media Sourcing - Pexels API (High-Resolution Photography)
PPTX Generation - PptxGenJS (Browser-side Library)
Core Logic - JavaScript (ES6+ / Async-Await / JSON Parsing)
Styling - CSS3 (Cyber-Grid Responsive Design)
Structure - HTML5 (Mobile-Optimized Layout)
Asset Fallback - Lorem Picsum (Dynamic Placeholder API)
